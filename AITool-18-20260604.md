**🛠 AI-ІНСТРУМЕНТ: Claude Code**

**Claude Code** — це CLI-інструмент від Anthropic, який запускається прямо в терміналі і працює з твоїм кодом, файлами, репозиторієм. Це не «чатик з ШІ», а агент, який реально редагує проєкти, запускає команди й тестує результат.

**Для кого корисний:**
- CEO та фаундери техстартапів — швидко перевірити гіпотезу без черги до розробників
- продакт-менеджери — зібрати прототип чи внутрішній скрипт самостійно
- маркетологи з технічним бекграундом — автоматизувати парсинг, звіти, інтеграції з API
- розробники-соло і tech-ліди невеликих команд

**У чому реальна різниця з ChatGPT:**
ChatGPT дає текст-відповідь, який ти копіюєш у редактор. Claude Code сам читає десятки файлів у проєкті, тримає контекст усієї кодової бази, вносить правки, запускає тести і виправляє помилки за результатом. Це різниця між «порадником» і «виконавцем».

**3 практичні сценарії:**

1. **Рефакторинг легасі.** Кажеш: «перепиши модуль авторизації під новий API, онови тести». Claude Code сам знаходить залежності у 20+ файлах і робить зміни. Результат: 4-6 годин роботи мідла за ~30 хвилин.

2. **MVP за вечір.** Описуєш ідею Telegram-бота для збору лідів з Google Sheets. Claude піднімає структуру, пише код, пояснює, як задеплоїти на Railway. Результат: робочий бот за 2-3 години без розробника.

3. **Аналітика по сирих даних.** Кидаєш CSV з продажів за рік — просиш знайти аномалії, побудувати графіки, написати звіт. Результат: готовий Python-скрипт + інсайти, які раніше робив аналітик за день.

**Скільки коштує:**
Працює в межах підписки **Claude Pro — $20/міс** (з лімітами) або **Max — від $100/міс** для активного використання. Є також оплата за API-токени (приблизно $3-15 за мільйон токенів залежно від моделі) — це для тих, хто хоче без лімітів.

**Як почати за 10 хвилин:**
1. Встанови Node.js
2. У терміналі: `npm install -g @anthropic-ai/claude-code`
3. Запусти `claude` у папці проєкту, авторизуйся
4. Перша команда: «прочитай проєкт і поясни мені архітектуру»

**Мінуси, які варто знати:**
- потрібен мінімальний комфорт із терміналом — не для повного «гуманітарія»
- на великих задачах токени горять швидко, Pro-підписки може не вистачати
- іноді «впевнено» робить неправильні архітектурні рішення — рев'ю обов'язкове

Claude Code — не заміна розробника, але реальний апгрейд швидкості для тих, хто вже щось робить руками.
